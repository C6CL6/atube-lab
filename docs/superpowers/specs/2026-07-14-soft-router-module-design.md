# Mac OS软路由模块设计

## 目标

用“Mac OS软路由”产品模块替代网站中的“创作手记”，介绍 A-tube 开发的 macOS 软路由，并提供公开 DMG 下载。

## 页面与导航

- 首页移除“创作手记”区块，改为“Mac OS软路由”入口。
- 顶部导航将“创作手记”替换为“软路由”，指向 `/router`。
- 新增 `/router` 页面，集中展示产品定位、核心能力、安装条件、下载信息和校验值。
- 删除创作手记详情路由；旧 `/notes/*` 路径回到首页，避免遗留失效页。

## 下载分发

- 分发文件：`Mac OS软路由.dmg`。
- 发布时将 DMG 放入 `public/downloads/`，让构建结果提供 `/downloads/Mac-OS软路由.dmg` 直链。
- 下载页展示 SHA-256：`c73338010ba9cbd1d0d3dd60c074365598fb61458a651c3ea0139cc66d4f81fd`。
- 页面说明安装包由 Developer ID 签名，并已通过 Apple 公证；不提示用户关闭 Gatekeeper 或修改系统安全设置。

## 文案范围

- 产品名称使用“Mac OS软路由”。
- 内容聚焦：将 Mac 作为家庭网络软路由、管理既有网络转发运行时、查看状态并执行启动/停止等需要管理员授权的操作。
- 明确运行要求为 macOS；不承诺绕过公司网络策略、地区限制或第三方服务限制。
